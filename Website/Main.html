ConcentrateAI

Code Jam Hackathon's Computer Vision Winner

December 2018
Github
  • Built with a team a web app which uses machine learning to help students study. The AI detects when students start to fall asleep or are dozing off and outputs audio/light waves to stimulate and wake them up.
  • The convolutional neural network was built using Python, TensorFlow, OpenCV-Python and Keras. It was trained using a database of 2423, 24 x 24 pixels pictures of people blinking for 10 epochs which gives an accuracy of 93.69%.
  • The web app was built using HTML5 and JavaScript.
